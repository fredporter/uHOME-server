# uHOME-server Getting Started

1. Review `docs/boundary.md`.
2. Review `docs/base-runtime-boundary.md`.
3. Review `docs/activation.md`.
4. Run `scripts/run-uhome-server-checks.sh`.
5. Keep service code under `services/` and `modules/`.
6. Route scheduling behavior through `scheduling/`.
7. Put new Matter or Home Assistant extension contracts in `uHOME-matter`.
8. Add tests before extending persistent behavior.
